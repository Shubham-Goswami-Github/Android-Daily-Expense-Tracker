# Finance Tracker App

A personal finance management Android app built with **Java + Firebase**, featuring:
- User authentication (Register/Login)
- Dashboard with analytics
- Income & expense tracking
- Personal details management
- Modern UI with Material Components

## 🚀 Features
- Firebase Authentication
- Firestore database
- Material UI Design
- Splash + Welcome Flow
- Dashboard with real-time tracking

## 📦 Tech Stack
- Java (Android)
- Firebase Auth
- Firebase Firestore
- Material Components
- XML UI
